# Kidney Diagnosis

A project to research and diagnose kidney disease using CNN.

## How to Run?

### Step 1: Clone the Repository

```bash
git clone https://github.com/susmitha-vengalasetty/kidney-diagnosis.git
cd kidney-diagnosis
```

### Step 2: Create a Conda Environment

```bash
conda create -n kidney python=3.10 -y
```

### Step 3: Activate the Environment

```bash
conda activate kidney
```


### Step 4: Install the Requirements

```bash
pip install -r requirements.txt
```

### Step 5: Install the Project Package

```bash
pip install -e .
```
